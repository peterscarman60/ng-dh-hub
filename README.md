# ng-dh-hub

## 项目简介

`ng-dh-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源存储与展示的通用空间。

## 目录说明

```
ng-dh-hub/
├── pages/           # 存放各独立 HTML 页面文件
├── assets/          # 页面所需的静态资源（图片、样式、脚本等）
├── index.html       # 仓库入口页面（可选）
└── README.md        # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，文件命名清晰以区分不同内容。
- `assets/`：按页面或功能组织，避免资源混乱。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无依赖关系。每个页面可直接通过浏览器打开运行，无需后端支持。

页面内容可能包含静态信息、工具界面或演示用途，不保证持续更新或兼容所有浏览器版本。如需引用或分发，请自行确认其适用性。
